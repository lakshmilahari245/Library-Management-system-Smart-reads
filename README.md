# 📚 SmartReads — Library Management System

A fully redesigned, modern Library Management System built with HTML, CSS, JavaScript, and localStorage.

## 🚀 Live Demo
Open `index.html` directly in any browser — **no server needed!**

Or visit the GitHub Pages link: `https://rajpra786.github.io/Library-Management-System/index.html`

## ✅ Features

### Admin Portal
- 📊 Dashboard with live stats (books, users, borrowed, available)
- 📚 Add, Edit, Delete books (full CRUD)
- 👥 View, inspect, and remove student accounts
- 🔍 Live search for books and students
- 📦 Buy Books — contact supplier page

### User Portal
- 🔐 Register and Login
- 📖 Browse complete book catalog
- 📤 Borrow available books
- ↩️ Return borrowed books
- 👤 Profile sidebar with borrowing history

## 📁 File Structure
```
├── index.html          # Home page (choose Admin or User)
├── admin_login.html    # Admin authentication
├── admin_portal.html   # Admin dashboard
├── add_book.html       # Add new book form
├── buy_book.html       # Supplier contacts
├── usr_login.html      # User login / signup
├── user_portal.html    # User book browsing
├── 404.html            # Not found page
├── css/
│   └── shared.css      # All styles (dark gold theme)
└── js/
    ├── data.js          # Shared localStorage data layer
    ├── admin_login.js   # Admin auth logic
    ├── admin_portal.js  # Dashboard CRUD logic
    ├── add_book.js      # Add book form logic
    ├── usr_login.js     # User login/register logic
    └── user_main.js     # User portal logic
```

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (ES6+)
- localStorage for data persistence
- Google Fonts (Playfair Display + DM Sans)
- No frameworks, no dependencies!
